# Device Testing Notes

This is a manual-install handheld port. Reports from other aarch64 devices are
welcome.

Please include:

- device model
- firmware / OS version
- Minecraft APK version
- whether first-run extraction completed
- whether graphics, audio, controls, touch, and LAN work
- relevant lines from `minecraftbedrock/log.txt` and
  `minecraftbedrock/weston_launch.log`

Do not upload APKs, extracted game files, worlds, `versions/`, `profiles/`, or
`libminecraftpe.so`.
